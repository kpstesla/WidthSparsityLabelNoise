# WidthSparsityLabelNoise
A framework for exploring the robustness to label noise of models with varied width and sparsity

## Supported Datasets
- CIFAR10
- CIFAR100
- MiniWebvision
- Clothing1M

## Supported Models
- CIFAR ReNets:
  - ResNet20
  - ResNet32
  - ResNet44
  - ResNet56
  - ResNet110
  - ResNet1202
- ImageNet ResNets:
  - ResNet18
  - ResNet34
  - ResNet50

## Supported Robust Methods
- ELR
- Mixup
